Open the Dev Tools console to use the calculator.

Access all methods and properties via the global "calculator" object.

Available Methods

  • .add(x, y) - Adds x and y. Saves sum in "currentNumber." If only one parameter is provided (e.g. calculator.add(2)), then the provided number is added to the current number (saved from last operation).
  • .subtract(x, y) - Subtracts y from x. Saves result in "currentNumber." If only one parameter is provided (e.g. calculator.subtract(2)), then the provided number is subtracted from the current number.
  • .multiply(x, y) - Multiplies x and y. Saves result in "currentNumber." If only one parameter is provided (e.g. calculator.multiply(2)), then the current number (saved from last operation) is multiplied by the provided number.
  • .divide(x, y) - Divides x by y. Saves result in "currentNumber." If only one parameter is provided (e.g. calculator.divide(2)), then the current number (saved from last operation) is divided by the provided number.
  • .modulo(x, y) - Returns the remainder of x divided by y and stores result in "currentNumber." If only one parameter is provided (e.g. calculator.modulo(2)), then it returns the remainder of dividing the current number (saved from last operation) by the provided number.
  • .modulo(x, y) - Returns the remainder of x divided by y and stores result in "currentNumber." If only one parameter is provided (e.g. calculator.modulo(2)), then it returns the remainder of dividing the current number (saved from last operation) by the provided number.
  • .master(equation) - Returns the evaluation of the equation. Nothing more, nothing less.
