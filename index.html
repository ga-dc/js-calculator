Calculator

Instructions:

Enter 'calculator.add(a,b)' in the console to add two numbers

Other methods available: subtract, multiply, divide, exponent, module, clear, master

Enter 2 parameters to perform a basic function

Enter 1 parameter to perform function on 'value' which starts at zero and then is saved into memory

calculator.clear() to clear memory value

pass any complex functions as a string into 'calculator.master()'
